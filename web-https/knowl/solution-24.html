Solution 21.2.1.

Solution: Since we denominator is zero when \(x=-2\) the domain is the set of all real numbers other than \(-2\text{.}\) As discussed in class, to find the range we solve the equatiuon
\begin{equation*} \frac{x+4}{x+2} = y \end{equation*}
for \(x\text{.}\) Multiplying with \((x+2)\) gives
\begin{equation*} x+4 = y(x+2) = yx + 2y. \end{equation*}
This can be reqwritten as
\begin{equation*} x-yx = 2y -4 \end{equation*}
or
\begin{equation*} (1-y)x = 2y-4. \end{equation*}
Thus
\begin{equation*} x = \frac{2y-4}{1-y} \end{equation*}
which is defined for all \(y\) except
\begin{equation*} y = 1. \end{equation*}
