## ww2-collegeAlgebra-exers

A compilation of the homework exerices from the [WeBWorK2's](https://webwork.maa.org/wiki/WeBWorK_Main_Page) [Model College Algebra Course](https://webwork.maa.org/wiki/ModelCourses/CollegeAlgebra).  Best accessed via GitHub Pages at [ww2-collegeAlgebra-exers](htt;s://justineuro.github.io/ww2-collegeAlgebra-exers).  

The syllabus and Sample pdf files of these homework exercises are available from the [MAA site](https://webwork.maa.org/wiki/ModelCourses/CollegeAlgebra) of this model course.
